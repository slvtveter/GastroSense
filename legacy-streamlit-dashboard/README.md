# Legacy: Streamlit dashboard

This was the original version of GastroSense — a single-file Streamlit app with baked-in synthetic presets, kept self-contained so it could be deployed for free without a separate backend.

The project has since migrated to a React + TypeScript frontend backed by a FastAPI service (see the repo root). That version has live model training, a real database, and a RAG-grounded Gemini chat assistant — things this Streamlit version never had.

This folder is kept for reference only and is **not deployed** anymore (it's no longer part of `render.yaml`). Nothing here is maintained going forward.
